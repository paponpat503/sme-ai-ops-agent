# SME Customer Onboarding Playbook

The standard onboarding flow has five steps:

1. Confirm business profile and primary use case.
2. Import customer records, product records, and workflow templates.
3. Configure CRM stages and automation triggers.
4. Test one end-to-end workflow with a sample customer.
5. Schedule a 30-minute review call after the first week.

If onboarding fails, the support team should identify whether the cause is missing data, incorrect field mapping, API authentication, or unclear workflow ownership.

For customers using AI sales agents, the team must verify prompt scope, knowledge-base documents, fallback messages, and human handoff conditions before production use.
